# Rasa Desa – Food Recommendation App UI/UX Prototype

Rasa Desa is a UI/UX prototype of a food recommendation application that helps users find traditional and local food (warung) based on their current location.

## 🔗 Prototype Link
https://www.figma.com/design/6YjuTt0aeASxmmtKJX572C/Untitled?node-id=0-1&m=dev&t=KB6pgFETJvypTj3y-1

## 📌 Features
- Find nearby traditional food (warung)
- Location-based recommendations
- View food and restaurant details
- Simple and user-friendly interface

- Customer / User
